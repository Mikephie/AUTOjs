关于本工具

这是一个通过 GitHub Actions 自动将 QuantumultX 格式的脚本转换为 Loon 和 Surge 格式的工具。所有脚本会定期自动更新，您可以直接下载使用或在线转换自己的脚本。

本项目开源于 GitHub，欢迎参与贡献或提出建议。
